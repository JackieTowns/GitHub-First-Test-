Git Commands

git init : intalizes the folder you want git to use ie in this case the Bootcamp folder

git status : get the git status of the file

git add : Add the file you want to commit

git commit : add the commit use "git commit -m "add file X" " to add a message. If you don't use the -m a text editor will appear and same thing.

git add . : use for committing everything in that place at one time and then use git commit to commit the combined files
